Meus Projetos

Projeto Runebooks

O RuneBooks foi o primeiro projeto completo que fiz, quando ainda estava começando a aprender desenvolver páginas. É um projeto que ainda engatinha, mas que já traz algumas funcionalidades. O site se trata de uma loja de livros, você tem a opção de compra ou detalhar sobre cada livro, vendo sinopse, autor etc. O site foi feito com apenas HTML e CSS puro.

Projeto Biomania

O projeto BioMania é um projeto de conteúdos sobre biologia, que traz textos de aprendizado sobre citologia. Foi o primeiro projeto em grupo que fiz, porém tive bastante contribuição para o projeto. Para a criação do site, foi utilizada HTML, CSS e o framework Bootstrap. Além do conteúdo, o site também conta com um joguinho em uma das páginas.

Jogo de Tiro ao Alvo

O projeto se trata de um jogo de tiro ao alvo com um sistema de pontuação, onde você precisa clicar no alvo (um gato) para contabilizar pontos. Nesse projeto foi quando tive o primeiro grande contato com Javascript e dinamização de páginas, que até então eram estáticas. Para o projeto foram utilizadas as tecnologias: HTML, CSS e Javascript.

Projeto Achaki

Se trata de uma rede social produzida para uma instituição pensada no uso dos alunos que perderam objetos. Esse projeto foi o primeiro que tive um grande foco em pesquisa, pois foi feito um estudo de campo e análise de design para que fosse feito. A página em questão, de linha do tempo, foi a minha maior contribuição pro projeto. Foram utilizados apenas HTML e CSS por se tratar de um projeto de design.
